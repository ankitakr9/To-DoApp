# To-DoApp
# 📝 To-Do List App

A simple and responsive To-Do List Application built using **HTML, CSS, and JavaScript**.  
This app allows users to add, delete, and manage their daily tasks efficiently.

---

## 🚀 Live Demo

🔗 https://to-do-app-tvcx.vercel.app/

---

## 📌 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Data stored in Local Storage
- 📱 Responsive design (Mobile + Desktop)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Vercel (Deployment)

## 💡 How It Works

1. Enter a task in the input field.
2. Click the Add button.
3. The task appears in the list.
4. You can mark it as complete or delete it.
5. Tasks remain saved even after refreshing the page.

---

## 📦 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repository-name.git
   ```

3. Open `index.html` in your browser.
